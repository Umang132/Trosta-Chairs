## 🔄 How to Update Products & Site Data

The website currently uses placeholder data for some products (items 5–12 reuse images from items 1–4). To update with real product data from **www.trostachairs.com**, follow these steps:

### Step 1: Provide Product Data

For each product, you need:
- **Name** — e.g. "Bubble", "Monarch"
- **Price** — in ₹ (number only, e.g. `13570`)
- **Category** — one of: `Executive Chairs`, `Ergonomic Chairs`, `Staff Chairs`, `Visitor Chairs`
- **Image URL** — upload to [imgbb.com](https://imgbb.com/) or any image host, then paste the direct link
- **Description** — a short 1-line description

You can provide this data by:
1. **Opening an issue** on this repository with the product details
2. **Commenting on the PR** with the data in the format shown in [`SITE_DATA_TEMPLATE.md`](SITE_DATA_TEMPLATE.md)
3. **Editing `index.html` directly** — find the `const products = [...]` array around line 1286 and update it

### Step 2: Provide Logo

Upload your logo image to [imgbb.com](https://imgbb.com/) and share the direct URL. Currently the site uses text-based branding ("TrostaChairs").

### Step 3: Provide Reviews

For each customer review, provide:
- **Customer name**
- **Star rating** (1–5)
- **Review text**
- **Company/role** (optional)

### Step 4: Provide Hero/Banner Images

Upload any hero section or banner images and share the direct URLs.

> **Tip:** The easiest way is to paste all this info into a GitHub Issue or PR comment, and we'll update the site for you!
